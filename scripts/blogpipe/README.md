Install from repo root: `pip install ./scripts` (see `setup.cfg`).

Run from repo root with: `BLOGPIPE_REPO_ROOT=$PWD python3 -m blogpipe run` after `PYTHONPATH=scripts` or pip install.

Commands: `curate`, `harvest`, `rank`, `research`, `draft`, `edit`, `visuals`, `package`, or `run` for the full chain.
